# CLI Mastery Track

Welcome to the CLI Mastery track. This track teaches you to become proficient with the VUDO command-line interface, the primary tool for DOL development.

## What is VUDO?

VUDO (Visionary Unified Development Operator) is the official CLI for DOL development. It provides a unified interface for:

- **Project scaffolding** - Create new projects from templates
- **Interactive development** - REPL for exploration and testing
- **Code quality** - Syntax checking, type validation, formatting
- **Registry operations** - Install and share DOL modules

```bash
# Quick install
cargo install vudo

# Verify installation
vudo --version
# vudo 0.2.3
```

## Quick Reference

| Command | Purpose |
|---------|---------|
| `vudo new` | Create new project from template |
| `vudo repl` | Start interactive REPL session |
| `vudo check` | Validate syntax and types |
| `vudo fmt` | Auto-format DOL files |
| `vudo doc` | Generate documentation |
| `vudo install` | Install a registry module |
| `vudo search` | Search the registry |
