## What You Will Build

The **Lion & Swan Creator** is a Spirit that:

- Defines mythological entities using DOL ontologies
- Generates AI-powered imagery via planetserve.io
- Creates narrative text passages for mythic scenes
- Manages credits for API consumption
- Provides a gallery interface for viewing content
