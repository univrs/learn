# DOL Fundamentals

Welcome to the DOL Fundamentals track! This series of 8 tutorials will take you from your first line of DOL code to building complete, type-safe systems with entities, error handling, and modular architecture.

## What You Will Learn

By the end of this track, you will be able to:

- Write and run DOL programs using the REPL
- Work with all primitive types and collections
- Define pure functions and closures
- Control program flow with expressions and pattern matching
- Create entities with components and relationships
- Handle errors gracefully with Result and Option types
- Organize code into reusable modules

## Prerequisites

No prior DOL experience is required! However, familiarity with any programming language will help you progress faster.

You should have:
- DOL installed on your system ([Installation Guide](/dol/reference/installation))
- A code editor with DOL syntax highlighting (optional but recommended)
- Basic understanding of programming concepts (variables, functions, etc.)

## How to Use These Tutorials

Each lesson includes:

1. **Introduction** - Context and learning objectives
2. **Interactive Examples** - Live REPL snippets you can modify and run
3. **Explanations** - Step-by-step breakdown of concepts
4. **Challenge** - A hands-on exercise to reinforce learning

We recommend completing the lessons in order, as each builds on concepts from previous tutorials.
